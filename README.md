# salesforce-cert-mock-exam
Salesforce Admin Mock Exam


## Temporary Public Access Mode
This build has PUBLIC_ACCESS_MODE enabled in app.py and practice/progress pages. Visitors can use the user-facing app without an Account login or certification enrollment. Admin pages still require ADMIN_PASSWORD in Streamlit Secrets.

To turn public mode off later, change PUBLIC_ACCESS_MODE = False in app.py, pages/Practice_By_Category.py, pages/Weak_Areas_Practice.py, pages/My_Progress.py, Practice_By_Category.py, Weak_Areas_Practice.py, and utils/access_control.py.
